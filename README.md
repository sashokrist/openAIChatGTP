
## About OpenAIChatGTP

Link to project: https://your-vibration.com/

To install and run the project follow these steps:

run in console:

git clone git@github.com:sashokrist/openAIChatGTP.git

cd openAIChatGTP

composer install

cp .env.example .env

php artisan key:generate

Set database credentials in .env need OPENAI_API_KEY form https://platform.openai.com/apps

php artisan migrate

php artisan serve
